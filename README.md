🚀 Nutraaafit – AI-Powered Nutrition & Preventive Health Platform

“What if you could see your future health—and change it today?”

🧠 Problem Statement

Modern lifestyles lead to poor dietary choices, increasing risks of chronic diseases like obesity, diabetes, and heart conditions. Most people don’t know what to eat even when they want to stay healthy.

💡 Solution

Nutraaafit is an AI-driven nutrition assistant that provides personalized diet recommendations based on user preferences, health conditions, and lifestyle to promote preventive healthcare.

✨ Features
🥗 Personalized diet plans
🧠 AI-based recommendations
📊 Health condition-based suggestions
🍛 Regional cuisine support
📝 Food logging system
🔍 Allergy & preference filtering

🛠️ Tech Stack
Frontend: React.js / JavaScript
Backend: Node.js, Express.js
Database: Supabase
AI: LLM-based recommendation system
Deployment: Lovable (Frontend), Replit/Vercel (Backend)
🧩 Architecture

User → Frontend → Backend → AI Layer → Database → Response

🔄 How It Works
User enters preferences, allergies, and goals
Backend processes data using AI
System generates meal recommendations
User tracks food and improves habits

📈 Future Scope
Wearable integration (smartwatch data)
Personalized diet auto-adjustment using AI
Predictive health analytics

⚙️ Setup
git clone https://github.com/akshayanilnair/Nutraaafit.git
cd Nutraaafit
npm install
npm start
